    Bienvenido a Chaucha Docs !

    Aquí encontrarás toda la información generada por el Proyecto Chaucha, que te ayudará a comprender el mundo de las criptomonedas de una manera fácil y sencilla.

    Como aportar

    Si piensas que algún tema presente en esta documentación es erroneo o quieres ayudarnos para mejorar este sitio, por favor visita el repositorio de Chaucha Docs en GitHub, en donde encontrarás el código fuente de este sistema.

    Los posts estan escritos en formato Markdown utilizando el framework MkDocs para compilar la versión web, que se encuentra alojada en Github Pages.

    Colaboradores
